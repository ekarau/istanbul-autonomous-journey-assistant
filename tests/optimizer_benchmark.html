Optimizer Benchmark

COE017 · compares registered optimizers against the greedy baseline on synthetic Istanbul scenarios.

How to plug in your optimizer

Bu benchmark window.Optimizers üzerinden çalışır. Optimization Specialist kendi modülünü yazınca aşağıdaki sözleşmeyi sağlamalı:

window.Optimizers = window.Optimizers || {};
window.Optimizers.simulatedAnnealing = function (scenario) {
    // scenario = { baseTimeMin, candidateFactorSets: string[][] }
    // return { chosenFactorSet: string[], iterations: number }
};

Greedy baseline (her senaryoda en düşük anlık E[D]'yi seçer) bu sayfada gömülü çünkü subject PDF benchmark için bir referans noktası gerektiriyor. Gerçek Hill Climbing / SA / GA / MCTS implementasyonu src/optimization/ altına ayrı dosya olarak gelecek.
